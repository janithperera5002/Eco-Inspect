# 🌍 Eco-Inspect: AI-Powered Environmental Monitoring

**Eco-Inspect** is an automated system designed to help local government environmental officers manage public complaints efficiently. Built for the **Airia AI Agents Hackathon**, it uses advanced AI to analyze WhatsApp reports in real-time.

## 🚀 The Inspiration
Inspired by my mother’s daily challenges as an Environment Officer, I developed this tool to automate the manual task of sorting through hundreds of complaints, ensuring critical hazards are prioritized instantly.

## 🛠️ Tech Stack
- **AI Orchestration:** [Airia AI Agents](https://airia.ai/) (Powered by Claude Haiku 4.5)
- **Backend:** Java Spring Boot, MySQL
- **Frontend:** React + Vite
- **Integration:** Twilio WhatsApp API, ngrok

## ✨ Key Features
- **WhatsApp Integration:** Citizens can report issues via a simple WhatsApp message.
- **AI Analysis:** Automatically extracts summaries, categories, and urgency levels from raw text.
- **Real-time Dashboard:** A web-based interface for officers to track and prioritize "Critical" reports.

## 📂 Project Structure
- `/eco-inspect-ai`: Spring Boot application handling API requests and AI integration.
- `/eco-inspect-ai_frontend`: React dashboard for data visualization.
